URL Shortener Application

Description:
This is a simple URL shrotener web application built with Flask and Python. The application allows users to convert
long URLs into short and easy to share links

How To Run:

Locally
Install pip install -r requiremnts.txt
Run python app.py
Access http://localhost:5000

Docker
Build: docker build -t url shortener
Run: docker run -p 5000:5000 url shortener
